# URL pública de CursosUmme

La web en producción está en **http://cursosumme.es** (no `.com`, no HTTPS todavía).

- Todas las URLs de test, fetch externas o curl deben usar `http://cursosumme.es`.
- El SSL está pendiente de configurar — una vez activo, cambiar a `https://` en `playwright.config.ts` y cualquier script.
